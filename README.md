# tp_regulation_gaelle_chaimae
 
- Kp faible (0.5) : la tortue tourne lentement
- Kp élevé (5.0) : la tortue oscille beaucoup
- Kp choisi (2.0) : bon compromis, mouvement stable


- Kpl = 0.5 → trop lent
- Kpl = 5 → oscillations importantes
- Kpl = 1.5 → comportement stable et rapide (choix retenu)
